# Hibrit Makro & Autoclicker Kontrol Paneli

Bu proje, hem **Görsel Tanıma (Image Recognition)** hem de **Sabit Koordinat** mantığıyla çalışan, gelişmiş ve hız ayarlı bir Python bot uygulamasıdır. 

## Özellikler
* **Görsel Odaklı Bot:** Ekrana verdiğiniz `.png` görseli geldiği an algılar ve kilitlenir.
* **Koordinat Odaklı Bot:** Belirlenen (X, Y) koordinatlarına periyodik tıklamalar gönderir.
* **Makro Hızı:** Ekran tarama darboğazı çözülmüştür; `0.01` saniye gibi yüksek makro hızlarına çıkabilir.
* **Sıralı Görev Listesi (Task List):** Birden fazla görevi farklı periyot ve tıklama adetleriyle sıraya koyabilirsiniz.
* **Acil Durdurma:** `F4` tuşuna basıldığı an bot tüm işlemleri güvenli bir şekilde durdurur.

## Kurulum ve Çalıştırma

1. Projeyi bilgisayarınıza indirin:
   ```bash
   git clone [https://github.com/KULLANICI_ADINIZ/REPA_ADINIZ.git](https://github.com/KULLANICI_ADINIZ/REPA_ADINIZ.git)
   cd REPA_ADINIZ

2. Gerekli kütüphaneleri yükleyin:

Bash
pip install -r requirements.txt
(Not: keyboard kütüphanesinin kısayolları algılayabilmesi için bazı sistemlerde terminali Yönetici/Administrator olarak çalıştırmanız gerekebilir).

3. Uygulamayı başlatın:

Bash
python main.py
Önemli Not
Bu araç eğitim ve otomasyon testleri amacıyla geliştirilmiştir. Çevrimiçi oyunlarda veya platformlarda kullanımı hesaplarınızın engellenmesine (ban) neden olabilir. Sorumluluk kullanıcıya aittir.
